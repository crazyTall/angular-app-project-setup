Grid

The main grid used is based on the Bootstrap grid system, but our classes have been renamed using the BEM naming convention. All grid columns should be wrapped in the 'row' class. Grid columns have a 10px padding on either side, resulting in a 20px gutter between columns and 10px gutter on grid edges.
